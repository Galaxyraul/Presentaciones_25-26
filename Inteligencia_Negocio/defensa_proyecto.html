Arquitectura de Descarga de Cuerpo de Noticias

Para los dominios con mayor volumen en Menéame (ej: El País, El Mundo, El Confidencial), se diseñan selectores CSS específicos para extraer el cuerpo neto de la noticia de forma impecable sin ruido (anuncios, navbars, sidebars).

Identificación de etiquetas HTML semánticas principales: <article>.
Recuperación exclusiva de texto limpio de párrafos editoriales: article p.
def extraer_el_pais(html_soup):
    # Captura exclusiva del artículo editorial
    article = html_soup.find('article')
    if article:
        parrafos = article.find_all('p')
        return " ".join([p.text.strip() for p in parrafos])

En diarios minoritarios o blogs donde no es eficiente programar scrapers manuales a medida, el sistema activa un **enfoque heurístico** inteligente.

Uso de la librería robusta de extracción periodística newspaper3k.
Fallback secundario basado en la **densidad de texto** de etiquetas HTML (contando ratio texto/enlaces en párrafos).
from newspaper import Article

def fallback_heuristico(url):
    art = Article(url, language='es')
    art.download()
    art.parse()
    return art.text

La descarga masiva requiere medidas defensivas críticas para evitar bloqueos por parte de los cortafuegos WAF de los periódicos.

Rotación de Cabeceras (User-Agent): Aleatorización de identidades de navegador real.
Retardos Estocásticos: Tiempos de pausa entre peticiones usando time.sleep(random.uniform(1, 3)).
Manejo de Errores:

Captura de excepciones HTTP (404, 403 de acceso prohibido) y caídas de red persistentes para evitar que falle el flujo de procesamiento masivo.
