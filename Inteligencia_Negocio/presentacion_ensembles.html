¿Qué son los Ensembles?

Combinando modelos para mayor precisión

  • Concepto: Técnica que combina múltiples modelos de aprendizaje para obtener un mejor rendimiento predictivo.
  • Idea Base: Un conjunto de aprendices débiles puede formar un aprendiz fuerte.
  • Estrategia: Promediar errores y capturar diferentes patrones de los datos.

04 / 11

Bagging (Bootstrap Aggregating)

Enfoque en Paralelo

  • Entrena múltiples versiones del mismo modelo con subconjuntos aleatorios.
  • Random Forest: El ejemplo estrella, usando múltiples árboles de decisión.
  • Ideal para modelos con alta varianza (sobreajuste).
05 / 11

Boosting

Enfoque Secuencial

  • Cada nuevo modelo intenta corregir los errores cometidos por los anteriores.
  • Modelos Clave: AdaBoost, Gradient Boosting, XGBoost, CatBoost.
  • Muy potente pero sensible al ruido y al sobreajuste si no se regula.
06 / 11

Stacking

Meta-Aprendizaje

  • Combina predicciones de modelos totalmente distintos (SVM, KNN, Redes Neuronales).
  • Un "meta-modelo" aprende a dar el peso adecuado a cada modelo base.
  • Arquitectura compleja: Modelos Base -> Meta-Modelo -> Predicción final.

Estado del Arte (SOTA)

Técnicas de vanguardia

  • TabPFN: El modelo definitivo para datos tabulares pequeños/medianos.
  • Deep Ensembles: Uso de ensembles en redes neuronales profundas para incertidumbre.
  • Kaggle Kings: Los ensembles de Boosting (XGBoost/LightGBM) dominan las competiciones.

Ventajas Técnicas

Compromiso Sesgo-Varianza

✓ Reduce la varianza sin aumentar mucho el sesgo (Bagging).

✓ Reduce el sesgo sin aumentar mucho la varianza (Boosting).

✓ Mayor robustez frente a valores atípicos y ruido en los datos.

10 / 11

Conclusiones

Conocimiento Accionable

  • Ningún modelo es perfecto, pero un conjunto de ellos se acerca.
  • Ensembles son el estándar de la industria para datos estructurados.
  • La clave es la diversidad entre los modelos del conjunto.
