Objetivos del Análisis

Exploración

Determinar si una variable categórica (como "Nivel de Estudios") segmenta eficazmente el comportamiento de una numérica (como "Salario").

Descubrimiento

  • • Identificar disparidades grupales.
  • • Detectar anomalías por categorías.
  • • Validar hipótesis de negocio/investigación.

Visualizaciones Combinadas

Barras

Representan la tendencia central (media) por categoría.

Boxplots

Muestran cuartiles, medianas y valores atípicos (outliers).

Violin Plots

Combinan boxplots con la densidad de probabilidad.

Validación: Tests de Hipótesis

z-test: Muestras grandes, varianza conocida. N > 30
t-test: Comparar medias de 2 grupos independientes. 2 Categorías
ANOVA: Comparar medias de 3 o más grupos. > 2 Categorías

Ecosistema de Análisis

Python Ecosystem

Pandas para manipulación, SciPy para tests y Seaborn para gráficos estadísticos de alta calidad.

Software BI & Data

Weka para minería de datos pura y Tableau para dashboards de segmentación bivariable.

Conclusiones Finales

  • La segmentación bivariable es el primer paso crítico para transformar datos en conocimiento accionable.

  • La validación estadística (p-valor) garantiza que los patrones encontrados no son producto del azar.

  • El uso de datasets reales como los de la UCI KDD permite entrenar modelos de minería con ruido y complejidad real.
