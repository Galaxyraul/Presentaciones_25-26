Los Segmentadores

Análisis Bivariable: Variables Numéricas vs. Categóricas

Relaciones Lineales Dependencias de Datos

¿Qué buscamos?

Objetivo Principal

Entender cómo una variable cualitativa (categoría) influye o segmenta el comportamiento de una variable cuantitativa (número).

Aplicación en KDD

  • • Perfilado de clientes (Edad vs. Segmento).
  • • Análisis de riesgos (Crédito vs. Nivel de Estudios).
  • • Segmentación de mercado.

Gráficos Combinados

Diagramas de Barras

Comparación de promedios o sumas por cada categoría.

Tartas (Pie Charts)

Visualización de proporciones relativas de la variable numérica.

Barras Apiladas

Descomposición de totales según sub-categorías.

Tip Pro: Para KDD, el Boxplot (Diagrama de Caja) es el rey para ver la distribución y outliers entre grupos.

Validación: Tipos de Test

Z

z-test

Usado para comparar medias cuando la muestra es grande (n > 30) y conocemos la desviación estándar poblacional.

T

t-test (Student)

Ideal para muestras pequeñas o cuando la varianza poblacional es desconocida. Evalúa si hay diferencia significativa entre dos medias.

A

ANOVA (Análisis de Varianza)

Se utiliza cuando comparamos medias de más de dos grupos simultáneamente.

Ejercicio Práctico

Repositorio: UCI KDD Archive

Ejercicio

Pasos del ejercicio:

  1. Seleccionar un dataset (Ej: Census-Income).
  2. Identificar variables Numéricas (Ej: Capital Gain) y Categóricas (Ej: Occupation).
  3. Realizar un t-test o ANOVA para ver si la ocupación influye en la ganancia.
  4. Visualizar la distribución mediante gráficos combinados.

Ecosistema de Herramientas

Python Stack

  • Pandas: Manipulación de dataframes.
  • Seaborn/Matplotlib: Generación de Boxplots y Bar charts.
  • SciPy / Statsmodels: Ejecución de t-test y ANOVA.

Otros

  • R Language: ggplot2 y tests base (t.test).
  • Tableau / PowerBI: Dashboarding interactivo.
  • Weka: Herramienta clásica de minería de datos (KDD).
